# BrandBuilder AI Agency Web Presence

This directory contains the web assets for the BrandBuilder AI Agency.

## Structure

- `index.html`: The main landing page for the agency.
- `portfolio.html`: Showcases sample work and link to deliverables.
- `intake-form.html`: The client intake form to capture brand requirements.
- `style.css`: Shared styles for all pages.

## Design Choices

- **Clean and Professional**: Uses a modern sans-serif font and a blue-dominant color palette to evoke trust and technology.
- **Responsive**: Built with a grid system and flexible containers to work across devices.
- **Action-Oriented**: Focuses on driving users to the intake form to start their branding journey.
